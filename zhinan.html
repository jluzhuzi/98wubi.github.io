新手指南：日常使用


Rime的配置方式为——文本文件配置，对初学者来说，修改难度很大。基于此，你下载到的输入法已经按最高效的方式配置过。


具体使用过程中，有一些快捷键你会需要的：


基础设定


1.回车 = 清屏


2.Shift + 回车 = 编码上屏


3.Shift = 中英切换


4.Ctrl + ~ = 切换方案


5.分号、引号 = 选 2、3 重码上屏


三大开关


Ctrl + Shift + H = 拆分提示开关


Ctrl + Shift + F = 繁简转换开关


Ctrl + Shift + U = 字符集过滤开关
